Slideshow examples

Favourites slideshow

$('#favorites-slideshow').sldshw(
	{
		action: 'favorites.getPublicList',
		userId: '61989727@N05' // http://www.flickr.com/people/thishappened/
	}
);

Javascript is required to load this slideshow from Flickr
